# Clone Netflix Site

> Build clone Netflix site use React, Tailwind CSS, Next.JS, Prisma, MongoDB, and NextAuth

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
